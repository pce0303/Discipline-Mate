# Discipline Mate

> 개인 맞춤 강의 추천 및 학습 계획 관리 웹 서비스

## 📌 프로젝트 소개

Discipline Mate는 사용자의 학습 목표와 일정을 관리할 수 있도록 돕는 학습 관리 웹 서비스입니다. 사용자는 로그인 후 자신의 학습 계획을 캘린더 형태로 확인하고, 필요한 계획을 추가하거나 관리할 수 있습니다. 단순히 일정을 기록하는 것을 넘어, 사용자의 학습 흐름을 정리하고 꾸준한 자기 관리를 돕는 것을 목표로 했습니다.

## 🛠 기술 스택

| 구분 | 기술 |
|---|---|
| Frontend | React |
| Backend | Express.js |
| Database | MySQL |
| Auth | Google OAuth |
| Tool | Git, GitHub |

## ✨ 주요 기능

- Google OAuth 기반 로그인
- 사용자별 학습 계획 관리
- 캘린더 기반 학습 일정 확인
- 학습 계획 생성 및 조회
- 프론트엔드와 백엔드 간 REST API 연동
- 사용자 정보 및 학습 데이터 DB 저장

## 🧩 담당 및 구현 내용

- React 기반 화면 구성
- 학습 계획 등록 및 조회 화면 구현
- Google OAuth를 활용한 로그인 흐름 구현
- Express 서버와 MySQL DB 연동
- REST API를 통해 사용자 정보와 학습 계획 데이터 연동
